# CHANGELOG: battle_analytics

Все изменения модуля аналитики боев фиксируются здесь.

---

### v0.2.1 (2026-05-14)
- **Сравнительный анализ:** Добавлен новый скрипт `generate_comparison_report.py` для детального сопоставления двух боев. 
- **Параллельный аудит:** Сравниваются отряды игрока и оппонента (включая генералов) с автоматической генерацией сводки изменений (уровни, звезды, заточки).
- **Исправление багов:** Устранен хардкод имен игроков в отчетах; исправлено определение имен генералов.
- **Документация:** В README добавлены инструкции по использованию новых инструментов генерации.

### v0.2.0 (2026-05-07)
- **Агрегация модулей:** Внедрен автоматический подсчет типов модулей и суммарных заточек в HTML-отчетах для юнитов и генералов.
- **Архитектура данных:** Переход на хранение исходных ("сырых") JSON-дампов. Аналитика вынесена в слой генерации отчетов.
- **Версионность дампов:** Реализовано сохранение ответов `/init` с временными метками для предотвращения потери истории.
- **Кодировка:** Исправлена обработка кириллицы и спецсимволов в именах игроков и путях к файлам.
- **Парсинг статов:** Улучшена логика очистки технических префиксов и суффиксов характеристик.
- **Визуализация:** Обновлен дизайн HTML-отчетов (цветовое выделение, лаконичные блоки данных).

### v0.1.0 (2026-04-28)
- Базовый функционал извлечения истории боев и генерации простых HTML-отчетов.
