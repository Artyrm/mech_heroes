# ORDA Clan Analytics - Style Guide (LOCKED)

Этот документ фиксирует визуальный стандарт дашборда клана ОРДА. Любые изменения в дизайне должны строго соответствовать этим правилам или быть затребованы явно.

## 1. Типографика (Шрифты)
- **Логотип (ORDA)**: шрифт `Orbitron`, кегль `4.5rem`, letter-spacing `12px`.
- **Основной текст и навигация**: шрифт `Inter`, кегль `16px`.
- **Цифровые данные (Рейтинг, Прирост)**: шрифт `Roboto Mono`, кегль `1.1rem` - `1.4rem`. 
- **Имена игроков**: шрифт `Inter`, жирный (700), кегль `1.2rem`.

## 2. Цветовая палитра
- **Фон (Background)**: `#0d1117` (темно-синий/черный).
- **Карточки (Cards)**: `#161b22`.
- **Границы (Borders)**: `#30363d`.
- **Акцент (Основной синий)**: `#58a6ff`.
- **Золото (Лидеры/Очки)**: `#f2cc60`.
- **Зеленый (Прирост)**: `#3fb950`.
- **Текст (Основной)**: `#c9d1d9`.

## 3. Структура Таблицы
1. **Header Row (Синий)**: Сводные итоги клана.
2. **TableHeader**: Названия колонок (№, Participant, Rank, Total Rating, Daily Growths).
3. **Rows**: Игроки, отсортированные по итоговому рейтингу, с порядковым номером (1, 2, 3...).

## 4. Поведение страницы
- **Скролл**: Вертикальная прокрутка страницы должна быть всегда доступна (`overflow-y: scroll`).

## 4. Логика данных
- **Колонка "Рейтинг"**: Текущая сумма очков.
- **Колонки дней**: Только **чистый прирост** за конкретный день.
- **Понедельник**: Прирост равен текущей сумме очков (так как старт недели с 0).
