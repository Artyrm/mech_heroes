# Clan Accountant Bot — История версий

> **Политика:** только **отлаженные и принятые в продакшен** версии.
> Экспериментальные ветки и черновые изменения сюда не вносятся.
> Текущая версия всегда дублируется в `VERSION_NUM` в начале `clan_accountant.py`.

---

### v0.2.6 (2026-04-30)
- **Метка времени в отчете:** В заголовок HTML-отчета добавлено время последнего успешного обновления данных («ОТЧЕТ СФОРМИРОВАН: DD.MM.YYYY HH:MM»).
- Улучшена логика вызова генератора отчетов для корректного отображения времени даже при сбоях.

### v0.2.5 (2026-04-30)
- **Защита от вылета сессии:** Добавлена функция `is_user_active()`. Бот проверяет наличие открытого окна игры и активного сетевого соединения с сервером перед выполнением `/init`.
- Если пользователь в игре, бот завершает работу без запросов к API, предотвращая разрыв игровой сессии.

### v0.2.4 (2026-04-29)

### Изменено
- Разделитель тысяч в HTML-выводе изменён с `,` на неразрывный пробел `\u00a0` (стандарт русской типографики). Введена вспомогательная функция `fmt(n: int) -> str` в начале скрипта.
- Знак отрицательного изменения рейтинга теперь явный (`-`), а не через `:,` с минусом — исправлено форматирование отрицательных значений в клановой статистике.

---

## [0.2.3] — 2026-04-28

### Добавлено
- **Auto-healing версии игры:** если сервер возвращает error `61058` («outdated version»), бот автоматически извлекает новый номер версии из текста ошибки, обновляет `config.json` и повторяет запрос без участия пользователя.
- **Manual Adjustments** (`manual_adjustments.json`): механизм ручных корректировок для игроков, покинувших клан в середине дня. Позволяет записать их очки на момент выхода, чтобы не потерять вклад.
- **Понедельная навигация** в дашборде: навигационная панель `<nav>` с переключением между неделями.
- **Детекция присутствия / ухода:** для каждого игрока по каждому дню определяется факт присутствия (был в последнем снэпшоте дня) или ухода (`X` в таблице).
- **Клановая статистика:** строка «ИСТОРИЧЕСКИЙ РЕЙТИНГ» с динамикой по дням и строка «СУММАРНЫЙ ЗАРАБОТОК». Отображаются «сожжённые» очки (прирост участников минус рост рейтинга клана).

### Изменено
- Разделитель тысяч в HTML-выводе изменён с `,` на неразрывный пробел `\u00a0` (русская типографика).

Версии до 0.2.3 не восстановлены из истории. Считаются архивными.
