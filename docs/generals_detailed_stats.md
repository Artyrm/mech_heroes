## Прокачка и стоимость (Leveling Costs)

Данные основаны на анализе `defs.json` (секция `generals.upgrade`) и верифицированы по игровым показателям.

### Механика уровней
1. **Структура:** Каждый уровень состоит из **4 шагов** (палочек): `.0, .1, .2, .3`.
2. **Переход:** Оплата последнего шага (например, `119.3`) переводит генерала на следующий уровень (`120.0`).
3. **Всего шагов:** Для достижения 120 уровня необходимо проплатить **476 шагов** (индексы 0-475 в массиве `levelCostCoefficients`).

### Формула расчета
Стоимость каждого шага вычисляется путем умножения коэффициента из `levelCostCoefficients` на фиксированный множитель:
*   **Опыт (Золотые звезды):** `Коэффициент * 3,5`
*   **Биты (Валюта):** `Коэффициент * 7,0`

*Пример (Уровень 4, шаг 0):*
Коэффициент `36,008` * 3,5 = **126 Опыта**
Коэффициент `36,008` * 7,0 = **252 Бита**
