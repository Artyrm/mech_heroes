# Система отслеживания и сравнения отрядов Арены

Этот функционал позволяет отслеживать эволюцию боевых отрядов игроков, находящихся в Топ-50 Арены, и сравнивать их составы между различными точками во времени.

## Как это работает

### 1. Сбор данных (`arena/fetch_squads.py`)
*   При каждом цикле обновления Арены (автоматическом или ручном) скрипт берет список ID всех игроков из актуального Топ-50.
*   Через эндпоинт `/directcommand` с типом `GetUsersRawInfos` запрашиваются детальные данные по каждому игроку.
*   Из ответа извлекается поле `squad` (содержит Генерала и список юнитов со всеми шмотками).
*   Вычисляется MD5-хэш строки отряда. Если он отличается от последнего сохраненного в `history.json` данного игрока, новый слепок добавляется в историю.

### 2. Генерация отчетов (`arena/generate_squad_reports.py`)
*   Для каждого игрока, по которому есть история отрядов, генерируется персональная HTML-страница в папке `arena/reports/squads/{userID}.html`.
*   Вся история изменений отряда "вшивается" прямо в эту страницу в формате JSON.

### 3. Визуализация и сравнение в браузере
*   На странице игрока доступны два выпадающих списка: **Точка T1 (Ранее)** и **Точка T2 (Сейчас)**.
*   При выборе точек JavaScript на клиенте мгновенно перерисовывает таблицу сравнения.
*   Юниты сопоставляются по их `defId`, что позволяет видеть эволюцию конкретного меха, даже если он переместился в другой слот.
*   Если в характеристиках юнита (уровень, звезды, уровень экипировки, заточки) есть различия, строка подсвечивается красным цветом (класс `diff`), и выводится текстовое резюме изменений.

## Как пользоваться

1.  Откройте главный дашборд Арены (`arena.html`).
2.  Кликните на ник интересующего вас игрока (они теперь оформлены как синие ссылки).
3.  Вы попадете на страницу отряда игрока.
4.  Выберите в выпадающих списках даты, которые вы хотите сравнить.
5.  Изучайте изменения: добавленные/удаленные юниты, рост уровней, смену модулей и заточек.

## Структура файлов
*   `arena/fetch_squads.py` — Скрипт сбора данных.
*   `arena/generate_squad_reports.py` — Генератор HTML-страниц.
*   `arena/squads/{userID}/history.json` — Сырые данные истории отрядов.
*   `arena/reports/squads/{userID}.html` — Страницы отчетов для просмотра в браузере.
